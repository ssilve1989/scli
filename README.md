# steve-cli

## Install

```bash
./install.sh
```

This installs dependencies, builds the project, and symlinks `scli` to `~/.local/bin/scli`.

## Development

Install dependencies:

```bash
bun install
```

Run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.9. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
